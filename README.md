# Efisio_trab_01

O programa criado funciona de forma que o usuário coloque como entrada as quantidades de VEs feitas, as respectivas notas e nota de VC. Como saída, o programa retorna a nota que usuário precisa tirar na VF para passar de ano. 

Para isso, o usuário digita o número de VEs, depois insere a nota correspondente ao número da VE, calculando a média de VE. Após isso, o usuário diz a nota de VC e é feito o cálculo para a nota de VF, da seguinte forma: (Média de VE)+ VC +VF=12.

Se a nota de VF for menor que 4, o aluno sabe que sua nota mínima para a VF será 4, caso contrário ele precisa estudar mais para alcançar a nota maior que 4.
